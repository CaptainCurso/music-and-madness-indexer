Privacy Policy

Music and Madness Scanner

This integration is intended for internal use with the Music and Madness Notion workspace.

What data the integration accesses

The integration reads content from the Music and Madness Notion wiki and reads rows from the M&M Index / Catalog database. It writes structured queue content to the Scanner Work Queue page inside the same Notion workspace.

How the data is used

Data is used only to discover pages that need indexing, detect duplicate catalog entries, and identify orphaned catalog rows. The tool does not use the content for advertising, profiling, or resale.

Data sharing

Data accessed by this integration is not intentionally shared with third parties except for the infrastructure required to run the tool and the Notion platform itself.

Retention

The scanner stores its source code in GitHub and writes queue output back into Notion. Any local logs or temporary runtime artifacts remain under the control of the project owner.

Contact

For questions about this integration, contact the repository owner through the associated GitHub account.
